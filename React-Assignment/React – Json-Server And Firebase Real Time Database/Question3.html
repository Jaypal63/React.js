How do you fetch data from a Json-server API in React? Explain the role of fetch() or axios() in making API requests.

Ans:

  • Fetching data from a JSON Server API in React involves making HTTP requests to retrieve or manipulate data hosted by the server.
  • The fetch() function and the axios() library are popular tools for making these requests.

    • Role of fetch() or axios()

    1. Send HTTP Requests (GET, POST, PUT, DELETE, etc.) to the server.
    2. Retrieve or Modify Data hosted by the JSON Server.
    3. Handle Responses (success or error) and pass the data to React components.
  • They act as the bridge between React app and the backend server, allowing dynamic interaction and real-time updates of your application's data.
